一、判断题（每题2分，共5题，10分）
1
函数 \(f(x) = \frac{1}{x}\) 在区间 \((0, +\infty)\) 内是有界函数。
答案： 错误
解析： 函数 \(f(x) = \frac{1}{x}\) 在区间 \((0, +\infty)\) 内是无界的，因为当 \(x \to 0^+\) 时，\(f(x) \to +\infty\)。
2
若 \(\lim_{x \to x_0} f(x)\) 存在，则 \(f(x)\) 在 \(x_0\) 处一定有定义。
答案： 错误
解析： 函数在某点的极限存在与否，并不要求函数在该点有定义。例如 \(f(x) = \frac{x^2-1}{x-1}\) 在 \(x=1\) 处极限为2，但该点无定义。
3
若函数 \(f(x)\) 在点 \(x_0\) 处可导，则 \(f(x)\) 在点 \(x_0\) 处一定连续。
答案： 正确
解析： 可导必连续，这是微分学的基本结论。函数在某点可导意味着该点处函数值的变化率存在，自然要求函数在该点连续。
4
函数 \(f(x) = |x|\) 在 \(x=0\) 处可导。
答案： 错误
解析： 函数 \(f(x) = |x|\) 在 \(x=0\) 处连续但不可导，因为左右导数不相等：\(f'_-(0) = -1\)，\(f'_+(0) = 1\)。
5
若函数 \(f(x)\) 在区间 \([a, b]\) 上连续，则它在该区间上一定可积。
答案： 正确
解析： 闭区间上的连续函数一定可积，这是定积分的基本性质之一。
二、填空题（每题3分，共5题，15分）
6
极限 \(\lim_{x \to 0} \frac{\sin 3x}{x} = \) ______
答案： 3
解析： 根据重要极限 \(\lim_{x \to 0} \frac{\sin x}{x} = 1\)，可得 \(\lim_{x \to 0} \frac{\sin 3x}{x} = \lim_{x \to 0} \frac{3\sin 3x}{3x} = 3\)。
7
函数 \(y = x^2 e^x\) 的导数 \(y' = \) ______
答案： \(2x e^x + x^2 e^x\) 或 \(x e^x (x+2)\)
解析： 使用乘积求导法则：\((uv)' = u'v + uv'\)，其中 \(u = x^2\)，\(v = e^x\)，则 \(y' = 2x \cdot e^x + x^2 \cdot e^x = x e^x (x+2)\)。
8
曲线 \(y = x^3 - 3x\) 在点 \((1, -2)\) 处的切线方程为 ______
答案： \(y = -2\)
解析： 先求导数：\(y' = 3x^2 - 3\)，在 \(x=1\) 处，\(y'(1) = 3(1)^2 - 3 = 0\)，所以切线斜率为0。切线方程为 \(y - (-2) = 0(x-1)\)，即 \(y = -2\)。
9
不定积分 \(\int (2x + \cos x) dx = \) ______
答案： \(x^2 + \sin x + C\)
解析： 根据基本积分公式：\(\int 2x dx = x^2\)，\(\int \cos x dx = \sin x\)，加上积分常数 \(C\)。
10
定积分 \(\int_0^{\pi/2} \cos x dx = \) ______
答案： 1
解析： \(\int_0^{\pi/2} \cos x dx = \sin x \big|_0^{\pi/2} = \sin(\pi/2) - \sin 0 = 1 - 0 = 1\)。
三、单选题（每题3分，共5题，15分）
11
函数 \(f(x) = \frac{x^2 - 1}{x - 1}\) 在 \(x = 1\) 处
A. 连续且可导
B. 连续但不可导
C. 不连续但可导
D. 不连续也不可导
答案： D
解析： 函数 \(f(x) = \frac{x^2 - 1}{x - 1} = x + 1\)（当 \(x \neq 1\)），且 \(\lim_{x \to 1} f(x) = 2\)，但 \(f(1)\) 无定义，所以不连续，因此也不可导，1是可去间断点，若补充定义 \(f(1) = 2\) 后函数在 \(x=1\) 处连续，也可导。
12
下列函数中，在区间 \((-1, 1)\) 内满足罗尔定理条件的是
A. \(f(x) = |x|\)
B. \(f(x) = \frac{1}{x}\)
C. \(f(x) = 1 - x^2\)
D. \(f(x) = x^3\)
答案： C
解析： 罗尔定理要求：1) 在闭区间上连续；2) 在开区间内可导；3) 区间端点函数值相等。选项C中，\(f(x) = 1 - x^2\) 在 \([-1, 1]\) 上连续，在 \((-1, 1)\) 内可导，且 \(f(-1) = f(1) = 0\)，满足所有条件。
13
设 \(y = \ln(1 + x^2)\)，则 \(d^2 y = \)
A. \(\frac{2}{1+x^2} dx^2\)
B. \(\frac{2(1-x^2)}{(1+x^2)^2} dx^2\)
C. \(\frac{2x}{1+x^2} dx^2\)
D. \(\frac{2x}{(1+x^2)^2} dx^2\)
答案： B
解析： 先求一阶微分：\(dy = \frac{2x}{1+x^2} dx\)，再求二阶微分：\(d^2 y = d(dy) = \left(\frac{2x}{1+x^2}\right)' dx^2 = \frac{2(1+x^2) - 2x \cdot 2x}{(1+x^2)^2} dx^2 = \frac{2(1-x^2)}{(1+x^2)^2} dx^2\)。
14
若 \(\int f(x) dx = x^2 + C\)，则 \(\int x f(1 - x^2) dx = \)
A. \(-\frac{1}{2}(1-x^2)^2 + C\)
B. \(\frac{1}{2}(1-x^2)^2 + C\)
C. \(-\frac{1}{4}(1-x^2)^2 + C\)
D. \(\frac{1}{4}(1-x^2)^2 + C\)
答案： A
解析： 由已知，\(f(x) = 2x\)。令 \(t = 1 - x^2\)，则 \(dt = -2x dx\)，所以 \(x dx = -\frac{1}{2} dt\)。代入得：\(\int x f(1 - x^2) dx = \int x \cdot 2(1-x^2) dx = 2\int x(1-x^2) dx = -\int t dt = -\frac{1}{2}t^2 + C = -\frac{1}{2}(1-x^2)^2 + C\)。
15
曲线 \(y = x^3 - 3x^2 + 2x\) 的拐点是
A. \((0, 0)\)
B. \((1, 0)\)
C. \((2, 0)\)
D. \((1, 0)\) 和 \((2, 0)\)
答案： B
解析： 求二阶导数：\(y' = 3x^2 - 6x + 2\)，\(y'' = 6x - 6\)。令 \(y'' = 0\)，得 \(x = 1\)。当 \(x < 1\) 时，\(y'' < 0\)；当 \(x > 1\) 时，\(y'' > 0\)，所以 \((1, f(1)) = (1, 0)\) 是拐点。
四、计算题（每题8分，共5题，40分）
16
计算极限 \(\lim_{x \to 0} \frac{e^{2x} - e^{-2x}}{\sin 3x}\)
答案： \(\frac{4}{3}\)
解析： \[ \begin{aligned} \lim_{x \to 0} \frac{e^{2x} - e^{-2x}}{\sin 3x} &= \lim_{x \to 0} \frac{2x \cdot \frac{e^{2x} - e^{-2x}}{2x}}{3x \cdot \frac{\sin 3x}{3x}} \\ &= \lim_{x \to 0} \frac{2x \cdot \frac{e^{2x} - e^{-2x}}{2x}}{3x \cdot \frac{\sin 3x}{3x}} \\ &= \frac{2}{3} \cdot \lim_{x \to 0} \frac{e^{2x} - e^{-2x}}{2x} \cdot \frac{1}{\lim_{x \to 0} \frac{\sin 3x}{3x}} \\ &= \frac{2}{3} \cdot (2) \cdot \frac{1}{1} = \frac{4}{3} \end{aligned} \] 或者使用洛必达法则：\(\lim_{x \to 0} \frac{e^{2x} - e^{-2x}}{\sin 3x} = \lim_{x \to 0} \frac{2e^{2x} + 2e^{-2x}}{3\cos 3x} = \frac{2+2}{3} = \frac{4}{3}\)。
17
求函数 \(y = \frac{x^2}{1+x}\) 的单调区间和极值
答案： 单调递增区间：\((-\infty, -2) \cup (0, +\infty)\)；单调递减区间：\((-2, -1) \cup (-1, 0)\)；极大值点：\((-2, -4)\)；极小值点：\((0, 0)\)
解析： \[ y' = \frac{2x(1+x) - x^2}{(1+x)^2} = \frac{2x + 2x^2 - x^2}{(1+x)^2} = \frac{x^2 + 2x}{(1+x)^2} = \frac{x(x+2)}{(1+x)^2} \] 令 \(y' = 0\)，得 \(x = 0\) 或 \(x = -2\)。函数定义域为 \(x \neq -1\)。
列表分析：
\(x\) \((-\infty, -2)\) \(-2\) \((-2, -1)\) \((-1, 0)\) \(0\) \((0, +\infty)\)
\(y'\) + 0 - - 0 +
\(y\) ↗ 极大值 ↘ ↘ 极小值 ↗
极大值：\(f(-2) = \frac{4}{-1} = -4\)；极小值：\(f(0) = 0\)。
18
计算不定积分 \(\int \frac{x}{\sqrt{1-x^2}} dx\)
答案： \(-\sqrt{1-x^2} + C\)
解析： 方法一：令 \(t = 1 - x^2\)，则 \(dt = -2x dx\)，所以 \(x dx = -\frac{1}{2} dt\)。 \[ \int \frac{x}{\sqrt{1-x^2}} dx = \int \frac{1}{\sqrt{t}} \cdot \left(-\frac{1}{2}\right) dt = -\frac{1}{2} \int t^{-1/2} dt = -\frac{1}{2} \cdot 2t^{1/2} + C = -\sqrt{t} + C = -\sqrt{1-x^2} + C \] 方法二：直接观察，因为 \(\left(\sqrt{1-x^2}\right)' = \frac{-x}{\sqrt{1-x^2}}\)，所以 \(\int \frac{x}{\sqrt{1-x^2}} dx = -\sqrt{1-x^2} + C\)。
19
计算定积分 \(\int_0^1 x e^{2x} dx\)
答案： \(\frac{e^2+1}{4}\)
解析： 使用分部积分法：令 \(u = x\)，\(dv = e^{2x} dx\)，则 \(du = dx\)，\(v = \frac{1}{2} e^{2x}\)。 \[ \begin{aligned} \int_0^1 x e^{2x} dx &= \left. \frac{1}{2} x e^{2x} \right|_0^1 - \int_0^1 \frac{1}{2} e^{2x} dx \\ &= \frac{1}{2} e^2 - \left. \frac{1}{4} e^{2x} \right|_0^1 \\ &= \frac{1}{2} e^2 - \frac{1}{4} e^2 + \frac{1}{4} \\ &= \frac{1}{4} e^2 + \frac{1}{4} = \frac{e^2+1}{4} \end{aligned} \]
20
求微分方程 \(y' + 2y = e^{-x}\) 的通解
答案： \(y = Ce^{-2x} + e^{-x}\)
解析： 这是一阶线性微分方程，标准形式为 \(y' + P(x)y = Q(x)\)，其中 \(P(x) = 2\)，\(Q(x) = e^{-x}\)。
通解公式：\(y = e^{-\int P(x) dx} \left[ \int Q(x) e^{\int P(x) dx} dx + C \right]\)。
计算：\(\int P(x) dx = \int 2 dx = 2x\)。
所以： \[ \begin{aligned} y &= e^{-2x} \left[ \int e^{-x} e^{2x} dx + C \right] \\ &= e^{-2x} \left[ \int e^{x} dx + C \right] \\ &= e^{-2x} (e^{x} + C) \\ &= e^{-x} + Ce^{-2x} \end{aligned} \]
五、证明题（10分）
21
证明：当 \(x > 0\) 时，\(\ln(1+x) > \frac{x}{1+x}\)
证明：
构造函数 \(f(x) = \ln(1+x) - \frac{x}{1+x}\)，则 \(f(0) = 0\)。
求导：\(f'(x) = \frac{1}{1+x} - \frac{1 \cdot (1+x) - x \cdot 1}{(1+x)^2} = \frac{1}{1+x} - \frac{1}{(1+x)^2} = \frac{x}{(1+x)^2}\)。
当 \(x > 0\) 时，\(f'(x) = \frac{x}{(1+x)^2} > 0\)，所以 \(f(x)\) 在 \((0, +\infty)\) 上单调递增。
因此，当 \(x > 0\) 时，\(f(x) > f(0) = 0\)，即 \(\ln(1+x) > \frac{x}{1+x}\)。证毕。
六、应用题（10分）
22
要建造一个容积为 \(V\) 的圆柱形无盖水池，问底半径 \(r\) 和高 \(h\) 各为多少时，才能使表面积最小？并求出最小表面积。
解：
已知圆柱体积公式：\(V = \pi r^2 h\)，所以 \(h = \frac{V}{\pi r^2}\)。
表面积（无盖）：\(S = \pi r^2 + 2\pi r h = \pi r^2 + 2\pi r \cdot \frac{V}{\pi r^2} = \pi r^2 + \frac{2V}{r}\)。
求导：\(S' = 2\pi r - \frac{2V}{r^2}\)。
令 \(S' = 0\)，得 \(2\pi r = \frac{2V}{r^2}\)，即 \(r^3 = \frac{V}{\pi}\)，所以 \(r = \sqrt[3]{\frac{V}{\pi}}\)。
此时 \(h = \frac{V}{\pi r^2} = \frac{V}{\pi \left(\frac{V}{\pi}\right)^{2/3}} = \left(\frac{V}{\pi}\right)^{1/3} = r\)。
即当底半径 \(r = \sqrt[3]{\frac{V}{\pi}}\)，高 \(h = \sqrt[3]{\frac{V}{\pi}}\) 时，表面积最小。
最小表面积：\(S_{\text{min}} = \pi r^2 + \frac{2V}{r} = \pi \left(\frac{V}{\pi}\right)^{2/3} + 2V \left(\frac{\pi}{V}\right)^{1/3} = 3\sqrt[3]{\pi V^2}\)。
答： 当底半径和高都等于 \(\sqrt[3]{\frac{V}{\pi}}\) 时，表面积最小，最小表面积为 \(3\sqrt[3]{\pi V^2}\)。
