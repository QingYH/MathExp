1. 常数项级数的收敛性定义、和函数、基本性质

大家好我是常数项级数Σan，我的梦想是收敛！收敛意味着我的部分和数列Sn有极限：

limn→∞Sn = S (存在)

我的和函数S(x)就是把我所有项加起来得到的函数~

⚠️ 警告：把我拆开重组可能会改变我的和值（条件收敛时），就像拆乐高重组可能变成抽象派作品

我的基本性质：

  • 线性性：Σ(kan+lbn) = kΣan + lΣbn
  • 增减有限项不改变敛散性（就像化妆改变不了基因）
  • 收敛的必要条件：lim an = 0（但这不是充分条件！调和级数警告）
2. 正项级数收敛性的花式判别法

我是正项级数（所有an≥ 0），判断我是否收敛的方法有：

部分和有界法：如果我的部分和Sn有上界，那我就收敛（有界的人生才有意义）

比较判别法：找个已知敛散性的好基友级数来比较大小

若 0 ≤ an ≤ bn，且Σbn收敛 ⇒ Σan收敛

比值判别法（达朗贝尔）：计算lim(an+1/an)

ρ < 1收敛，ρ > 1发散，ρ=1可能翻车

根值判别法（柯西）：计算lim ⁿ√an

ρ < 1收敛，ρ > 1发散，ρ=1可能翻车

🤔 比值和根值判别法就像相亲：快速筛选但不保证100%准确

3. 常用级数的敛散性（明星级数天团）

等比级数 Σarⁿ⁻¹：当|r| < 1时收敛，和S=a/(1-r)

调和级数 Σ1/n：著名的发散级数（虽然项趋近于0）

p级数 Σ1/nᵖ：p>1时收敛，p≤1时发散

🎸 调和级数是摇滚明星——永远躁动不收敛！

4. 交错级数の莱布尼兹判别法

我是交错级数Σ(-1)ⁿ⁻¹an（an> 0），要判断我是否收敛：

  1. an单调递减（越来越小）
  2. lim an= 0

同时满足 ⇒ 我收敛！

💃 交错级数就像跳踢踏舞：左右摇摆但整体趋势向下

5. 绝对收敛 vs 条件收敛

绝对收敛：Σ|an|收敛 ⇒ Σan必收敛（霸道总裁型）

条件收敛：Σan收敛但Σ|an|发散（傲娇型）

⚠️ 条件收敛的级数像渣男：重排可能改变和值（黎曼重排定理）

6. 幂级数的收敛性（阿贝尔定理）

我是幂级数Σanxⁿ，我的收敛性由阿贝尔定理决定：

如果在x=x₀收敛 ⇒ 对∀|x| < |x₀|都绝对收敛

如果在x=x₀发散 ⇒ 对∀|x| > |x₀|都发散

🔍 阿贝尔定理就像探照灯：找到一个收敛点就能照亮一片区域

7. 幂级数收敛半径的求法

我的收敛半径R可以通过：

R = 1 / lim sup ⁿ√|an| （柯西-阿达马公式）

或当极限存在时：

R = lim |an/an+1|

收敛区间：( -R, R )

收敛域：要单独检查端点x=±R

📏 收敛半径就像安全距离：超出这个范围关系就会发散~

8. 幂级数的运算与和函数

我可以进行以下操作：

逐项求导：(Σanxⁿ)' = Σnanxⁿ⁻¹（收敛半径不变）

逐项积分：∫(Σanxⁿ)dx = Σ(anxⁿ⁺¹)/(n+1) + C（收敛半径不变）

求和函数的常用方法：

  • 利用已知幂级数展开式
  • 逐项求导或积分
  • 解微分方程

🧮 逐项操作就像乐高：拆开操作再拼回去

9. 函数展开为幂级数

把函数f(x)展开成幂级数的两种方法：

直接法：计算f⁽ⁿ⁾(0)/n!（泰勒级数）

间接法：利用已知展开式进行变量替换、积分、求导等操作

🎁 常见展开式（记不住就完蛋系列）：

eˣ = Σxⁿ/n!
sinx = Σ(-1)ⁿx²ⁿ⁺¹/(2n+1)!
cosx = Σ(-1)ⁿx²ⁿ/(2n)!
1/(1-x) = Σxⁿ (|x|<1)
10. 欧拉公式：数学界の最美公式
eⁱˣ = cosx + isinx

特别地，当x=π时：

eⁱπ + 1 = 0

这个公式连接了五个最重要的数学常数！

💖 欧拉公式就像数学界的莎士比亚十四行诗

11. 傅里叶级数：振动分解大师

周期为2π的函数f(x)可以展开为：

f(x) ∼ a₀/2 + Σ(ancosnx + bnsinnx)
其中 an = (1/π)∫f(x)cosnx dx
bn = (1/π)∫f(x)sinnx dx

狄利克雷条件：

  1. 周期内绝对可积
  2. 有限个极值点
  3. 有限个第一类间断点

🎵 傅里叶级数就像音乐合成器：把复杂波形分解成纯音

📝 级数知识大闯关（模拟试题）

1.判断题（每题2分）

(1). 若Σan收敛，则Σan²必收敛。（ ）

答案：×（反例：an= (-1)ⁿ/√n）

(2). 幂级数的收敛域一定是区间。（ ）

答案：×（可能是单点或整个实数域）

2.填空题（每题3分）

(1). 级数Σ(1/2ⁿ)的和是______。

答案：1

(2). 幂级数Σxⁿ/n的收敛半径R=______。

答案：1

3.选择题（每题4分）

(1). 下列级数中收敛的是：

A. Σ1/n B. Σ1/√n C. Σ1/n² D. Σ1/lnn

答案：C

(2). 若lim|an+1/an|=2，则Σanxⁿ的收敛半径是：

A. 1/2 B. 2 C. ∞ D. 0

答案：A

4.计算题（每题10分）

(1). 判断Σn!/nⁿ的敛散性。

答案：使用比值判别法，lim|an+1/an|=1/e < 1，故收敛

(2). 求幂级数Σ(-1)ⁿx²ⁿ⁺¹/(2n+1)的收敛域。

答案：收敛半径R=1，收敛域[-1,1]

5.应用题（12分）

利用幂级数展开计算∫e⁻ˣ² dx在x=0.5处的近似值（取前三项）。

答案：e⁻ˣ²≈1-x²+x⁴/2，积分得≈0.5-0.5³/3+0.5⁵/10≈0.4615

6、判断题（每题3分）
(1). 若级数Σan收敛，则必有lim an = 0。
正确。这是级数收敛的必要条件。
(2). 正项级数Σan收敛的充要条件是其部分和数列有界。
正确。这是正项级数收敛的基本性质。
(3). 若Σ|an|发散，则Σan必发散。
错误。例如交错调和级数Σ(-1)ⁿ/n条件收敛但非绝对收敛。
7、填空题（每题4分）
(1). 级数Σ(1/2)ⁿ的和为______。
1。这是首项为1/2，公比为1/2的等比级数。
(2). 幂级数Σxⁿ/n的收敛半径R=______。
1。使用比值法lim|an+1/an|=1。
(3). 函数f(x)=1/(1+x)在x=0处的泰勒级数前3项为______。
1 - x + x²。由1/(1+x)=Σ(-1)ⁿxⁿ。
8、选择题（每题5分）
(1). 下列级数中收敛的是：

A. Σ1/n    B. Σ1/√n    C. Σ1/n²    D. Σ1/lnn

C。A、B是发散的p级数(p≤1)，D与调和级数比较可知发散。
(2). 若lim ⁿ√|an|=1/2，则幂级数Σanxⁿ的收敛半径为：

A. 1/2    B. 2    C. ∞    D. 0

B。根据根值法R=1/(1/2)=2。
(3). 下列哪个不是傅里叶级数的狄利克雷条件：

A. 周期内绝对可积    B. 有限个极值点

C. 处处连续    D. 有限个第一类间断点

C。傅里叶级数允许有限个第一类间断点。
9、计算题（每题10分）
(1). 判断级数Σn!/nⁿ的敛散性。
使用比值判别法：lim|an+1/an|=lim(1+1/n)⁻ⁿ=1/e<1，故收敛。
(2). 求幂级数Σ(-1)ⁿx²ⁿ⁺¹/(2n+1)的收敛域。
收敛半径R=1，端点x=1收敛(莱布尼兹判别法)，x=-1也收敛，故收敛域[-1,1]。
