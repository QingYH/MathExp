🔷 二次曲面标准库 (空间解析几何)

椭球面 x^2/4 + y^2/9 + z^2/16 = 1 三轴不等椭球，输入 x^2 + y^2 + z^2 = 9 球面

单叶双曲面 x^2/4 + y^2/9 - z^2/16 = 1 冷却塔形状 | 双叶双曲面 x^2/4 - y^2/9 - z^2/16 = 1

椭圆抛物面 z = x^2/4 + y^2/9 (开口向上) | 双曲抛物面(马鞍面) z = x^2/4 - y^2/9

椭圆锥面 z^2 = x^2/4 + y^2/9 或 x^2/4 + y^2/9 = z^2 | 椭圆柱面 x^2/4 + y^2/9 = 1 (无限延伸柱体)

🎨 参数曲面演示

  • 环面(Torus): 曲面((2+cos(u))cos(v), (2+cos(u))sin(v), sin(u), u, 0, 2π, v, 0, 2π)
  • 螺旋曲面: 曲面(u cos(v), u sin(v), v, u, 0, 2, v, 0, 4π)
  • 莫比乌斯带: 曲面((1+0.5u cos(v/2))cos(v), (1+0.5u cos(v/2))sin(v), 0.5u sin(v/2), u, -0.5, 0.5, v, 0, 2π)
💡 交互技巧：在3D模式下，右键曲面可修改透明度，便于观察内部结构。使用 平面(z=0) 截取曲面得到交线。

📦 多重积分区域 · 体积分域

直角坐标二重积分区域 矩形域: 多边形((0,0,0), (2,0,0), (2,3,0), (0,3,0)) 投影到xOy平面，再用 棱柱 或 曲面 构建立体。

X型/Y型区域 输入 不等式(0 ≤ y ≤ x^2, x, 0, 2) 绘制2D区域，3D中拉伸: 曲面((x, y, t), t, 0, f(x,y)) 构建立体。

柱坐标积分区域 圆柱体: 曲面(r cos(θ), r sin(θ), z, r, 0, 2, θ, 0, 2π, z, 0, 3) 圆柱区域。圆锥: 曲面(r cos(θ), r sin(θ), r, r, 0, 2, θ, 0, 2π)

球坐标积分区域 球体: 曲面(ρ sin(φ)cos(θ), ρ sin(φ)sin(θ), ρ cos(φ), ρ, 0, 2, φ, 0, π, θ, 0, 2π)；球扇形区域限定φ范围。

🔥 典型三重积分域案例

  • 抛物面与平面围域: z = x^2 + y^2 和 z = 4 之间的区域 → 使用 相交路径(曲面, 平面) 得到交线圆。
  • 圆柱与锥体组合: x^2 + y^2 ≤ 4, |z| ≤ sqrt(x^2+y^2) 双锥区域。
  • 球体与圆柱体相交(Viviani区域): x^2+y^2+z^2 ≤ 4 且 x^2+y^2 ≤ 2x，用 相交路径 展示交线。
📌 积分域绘制命令：区域(x^2+y^2 ≤ 4, z ≥ 0, z ≤ 4-x^2-y^2) 直接绘制三维立体区域。

🔄 旋转曲面 & 投影区域可视化

绕x轴旋转曲面 曲面(u, f(u)cos(v), f(u)sin(v), u, a, b, v, 0, 2π) 例如 f(u)=u^2 得旋转抛物面。

绕y轴旋转曲面 曲面(f(u)cos(v), u, f(u)sin(v), u, a, b, v, 0, 2π) 绕y轴。

空间曲线投影 输入曲线 c: 曲线(cos(t), sin(t), t, t, 0, 4π)，投影到xOy平面: 垂线(c, xOy平面) 得到投影曲线。

二重积分投影区域 从立体向坐标平面投影: 使用 相交曲线(曲面, z=0) 得到底面边界，再用 多边形 或 区域 填充。

📐 极坐标积分区域示例

  • 心形线区域: 曲线(2+2cos(θ), θ, 0, 2π) 极坐标曲线，内部区域用 区域(0 ≤ r ≤ 2+2cos(θ))
  • 圆环区域: 区域(1 ≤ r ≤ 2) 极坐标环形。
  • 玫瑰线区域: 曲线(3cos(2θ), θ, 0, 2π) 四叶玫瑰线包围区域。
🧠 教学建议：使用滑动条动态展示旋转体生成过程: v=滑动条(0, 2π, 0.1) 构建部分旋转体。

✂️ 曲面交线 · 积分边界曲线

圆柱与平面交线 x^2+y^2=4 与 z=x+2 相交得椭圆曲线: 相交曲线(圆柱, 平面)

球面与锥面交线 x^2+y^2+z^2=9 与 z^2=x^2+y^2 相交得两个圆。

参数化交线用于线积分 交线可提取为 曲线 对象，计算 长度(曲线) 或线积分。

截面切片动画 创建滑动条 c=滑动条(-3,3,0.1)，输入 平面(z=c) 动态观察曲面截面变化，理解二重积分截面法。

🎯 曲面积分投影区域案例

  • 曲面 z = x^2 + y^2 在xOy投影: 直接观察底面圆域。
  • 用 曲面(x, y, f(x,y), x, -2, 2, y, -2, 2) 并设置透明度，投影区域用 区域(x^2+y^2 ≤ 4) 突出显示。
  • 向量场通过曲面的通量演示: 使用 向量场( (x, y, z) ) 结合曲面。
🌟 高级应用：使用 相交路径(球面, 圆柱) 得到维维亚尼曲线，用于线积分教学。

🧮 多重积分数值计算 (CAS)

二重积分计算 在CAS视图输入: 积分(积分(x^2+y^2, x, -1, 1), y, -1, 1) 得矩形域积分。

极坐标二重积分 积分(积分(r^2 * r, r, 0, 2), θ, 0, 2π) 计算圆域上 x^2+y^2 积分。

三重积分(直角坐标) 积分(积分(积分(x+y+z, z, 0, 1-x-y), y, 0, 1-x), x, 0, 1) 四面体区域。

柱坐标三重积分 积分(积分(积分(r * r, z, 0, 4-r^2), r, 0, 2), θ, 0, 2π) 计算抛物面体积。

📊 体积计算可视化验证

  • 球体体积: 体积(x^2+y^2+z^2 ≤ 9) 直接计算数值。
  • 旋转体体积: 体积(曲面(u, f(u)cos(v), f(u)sin(v), u, 0, 2, v, 0, 2π)) 得到数值结果。
  • 两曲面之间体积: 体积(曲面1, 曲面2, 投影区域) 快速计算。

📚 综合案例 & GeoGebra 操作指南

案例1: 椭球与平面相交求截面面积 输入 椭球: x^2/9+y^2/4+z^2=1，平面: x+y+z=1，相交曲线(椭球, 平面) 得到椭圆，右键测量面积。

案例2: 圆柱与抛物面包围区域体积 区域: 区域(x^2+y^2 ≤ 4, z ≥ 0, z ≤ 4-x^2-y^2)，右键属性可查看体积近似值。

案例3: 参数曲面族的动画演示 创建滑动条 k=滑动条(0,2,0.01)，输入 曲面(k*u*cos(v), k*u*sin(v), u^2, u, 0, 2, v, 0, 2π) 动态生长曲面。
