曲线曲面积分の欢乐课堂

🎪 1. 对弧长的曲线积分 - "我是曲线界的体重秤"

定义：就像给弯弯曲曲的铁丝称体重，∫L f(x,y) ds表示沿着曲线L给函数f(x,y)称重！

性质：

  • 线性性：可以分开称（∫(af+bg)ds = a∫fds + b∫gds）
  • 与方向无关：正着走反着走重量不变
  • 可加性：把曲线分段称，再相加
🧮 2. 第一类曲线积分计算方法 - "参数化变身术"

步骤口诀：

  1. 把曲线L用参数t表示：x=x(t), y=y(t), t∈[α,β]
  2. 计算弧微分：ds = √[(dx/dt)² + (dy/dt)²] dt
  3. 把f(x,y)中的x,y换成t的表达式
  4. 积分：∫αβ f(x(t),y(t))·√[x'(t)²+y'(t)²] dt
🚨 注意：一定要先参数化！直接积分会被数学老师打手心！
🧭 3. 第二类曲线积分 - "有方向的能量消耗"

定义：∫L Pdx + Qdy 表示沿着有向曲线L做功的总量！

与第一类的区别：

  • 有方向性！反向积分要变号
  • 物理意义：计算力场做功

计算方法：

  1. 参数化曲线：x=x(t), y=y(t), t从α到β
  2. 把dx,dy表示为dt：dx=x'(t)dt, dy=y'(t)dt
  3. 积分：∫αβ [P(x(t),y(t))x'(t) + Q(x(t),y(t))y'(t)] dt
🤝 4. 两类曲线积分の联谊会 + 格林公式

联系公式：∫L Pdx+Qdy = ∫L (Pcosα + Qcosβ) ds

格林公式：∮∂D Pdx+Qdy = ∬D (∂Q/∂x - ∂P/∂y) dxdy

应用场景：

  • 计算封闭曲线积分时偷懒用
  • 判断积分与路径无关的条件（∂Q/∂x = ∂P/∂y）
🚨 格林公式使用条件：
1. 封闭曲线正向（逆时针）
2. P,Q在D内连续可微
3. 遇到奇点要挖洞！
🍕 5. 第一类曲面积分 - "曲面披萨的面积称重"

定义：∬Σ f(x,y,z) dS 给曲面Σ上的函数称重

计算方法：

  1. 投影法：把曲面投影到坐标平面（通常选z=z(x,y)）
  2. 计算面积元素：dS = √[1 + (∂z/∂x)² + (∂z/∂y)²] dxdy
  3. 转化为二重积分：∬D f(x,y,z(x,y))·√[1+zx²+zy²] dxdy
🧲 6. 第二类曲面积分 - "有方向的流量计算"

定义：∬Σ Pdydz + Qdzdx + Rdxdy 表示通过曲面Σ的流量

计算方法：

  1. 确定曲面侧（法向量方向）
  2. 投影到坐标平面（如z=z(x,y)）
  3. 转化为二重积分：±∬D [P(-∂z/∂x) + Q(-∂z/∂y) + R] dxdy
  4. 正负号由曲面侧决定
🔗 7. 两类曲面积分の联谊会 + 高斯公式

联系公式：∬Σ Pdydz+Qdzdx+Rdxdy = ∬Σ (Pcosα + Qcosβ + Rcosγ) dS

高斯公式：∯∂Ω Pdydz+Qdzdx+Rdxdy = ∭Ω (∂P/∂x + ∂Q/∂y + ∂R/∂z) dxdydz

应用场景：

  • 计算封闭曲面积分时偷懒用
  • 计算空间区域的体积（令P=x,Q=y,R=z）
🌀 8. 斯托克斯公式 - "格林公式の空中转体360度"

公式：∮∂Σ Pdx+Qdy+Rdz = ∬Σ (∂R/∂y-∂Q/∂z)dydz + (∂P/∂z-∂R/∂x)dzdx + (∂Q/∂x-∂P/∂y)dxdy

记忆口诀：
"RQ差对y和z，PR差对z和x，QP差对x和y"

应用场景：空间曲线积分转化为曲面积分

🔄 9. 对称性の偷懒大法

各类积分中的对称性应用：

  • 偶函数：对称区间积分可以×2
  • 奇函数：对称区间积分直接=0
  • 轮换对称性：x↔y↔z互换积分值不变

使用技巧：

  1. 先观察积分区域的对称性
  2. 再看被积函数的奇偶性
  3. 能简化计算就简化，不能就老实算

2. (填空题) 计算曲线积分∫L xyds，其中L是抛物线y=x²从(0,0)到(1,1)的弧段，第一步参数化应设为x=____, y=____。 t, t² (t∈[0,1])

3. (选择题) 格林公式成立的条件不包括：
A) 封闭曲线 B) P,Q连续可微 C) 逆时针方向 D) 曲面光滑 D

4. (计算题) 计算∬Σ z dS，其中Σ是球面x²+y²+z²=a²的上半部分。
解：参数化后用极坐标计算，结果为πa³

5. (应用题) 用高斯公式计算∯Σ xdydz + ydzdx + zdxdy，Σ是球面x²+y²+z²=R²的外侧。
解：由高斯公式得3×(4/3πR³)=4πR³

6. (证明题) 证明当∂Q/∂x=∂P/∂y时，曲线积分∫L Pdx+Qdy与路径无关。
提示：用格林公式和路径无关的等价条件

7. 计算第一类曲线积分：求∫L (x+y)ds，其中L为从(0,0)到(1,1)的直线段。

解：参数化直线为x=t, y=t, t∈[0,1]
ds=√(1+1)dt=√2 dt
∫01 (t+t)·√2 dt = √2∫01 2t dt = √2[t²]01 = √2

8. 计算第二类曲线积分：求∫L ydx - xdy，其中L为上半圆周x²+y²=1从(1,0)到(-1,0)。

解：参数化为x=cost, y=sint, t∈[0,π]
dx=-sint dt, dy=cost dt
∫0π [sint·(-sint) - cost·cost]dt = -∫0π (sin²t+cos²t)dt = -∫0π 1 dt = -π

9. 第一类曲面积分：计算∬Σ z dS，其中Σ为平面z=1被x²+y²≤1截下的部分。

解：dS=√(0+0+1)dxdy=dxdy
∬D 1 dxdy = π·1² = π (D为单位圆)

10. 第二类曲面积分：计算∬Σ xdydz + ydzdx + zdxdy，Σ为立方体0≤x,y,z≤1的表面外侧。

解：用高斯公式
∭Ω (1+1+1)dxdydz = 3×1 = 3

11.（格林公式）：计算∮L (x²-y²)dx + (2xy)dy，其中L是矩形区域0≤x≤2，0≤y≤1的边界正向。

解：应用格林公式
P=x²-y², Q=2xy
∂Q/∂x=2y, ∂P/∂y=-2y
∬D (2y-(-2y))dxdy = 4∬D y dxdy
=4∫02∫01 y dydx = 4×2×[y²/2]01 = 4

12.（格林公式）：验证格林公式对∮L (eˣcosy)dx + (eˣsiny)dy，其中L是三角形顶点(0,0),(1,0),(1,1)的正向边界。

解：格林公式右边
∂Q/∂x=eˣsiny, ∂P/∂y=-eˣsiny
∬D (eˣsiny-(-eˣsiny))dxdy = 2∬D eˣsiny dxdy
=2∫01∫0x eˣsiny dydx = ... = e(1-cos1)-1

13.（格林公式）：计算∮L (arctan(y/x))dx + (ln(x²+y²))dy，其中L是圆周x²+y²=4逆时针方向。

解：注意(0,0)是奇点，需挖去小圆ε²
格林公式得：
∮L - ∮lε = ∬Dε (2x/(x²+y²)-(-y/(x²+y²)))dxdy = 3π
计算小圆积分得2π
最终结果=3π-2π=π

14.（高斯公式）：计算∯Σ (x³)dydz + (y³)dzdx + (z³)dxdy，其中Σ是球面x²+y²+z²=R²外侧。

解：应用高斯公式
∭Ω (3x²+3y²+3z²)dxdydz
球坐标变换：
=3∫02π∫0π∫0R r²·r²sinφ drdφdθ
=3×2π×2×(R⁵/5) = (12πR⁵)/5

15. 高斯公式应用：用高斯公式计算∯Σ x²dydz + y²dzdx + z²dxdy，Σ为球面x²+y²+z²=R²外侧。

解：∭Ω (2x+2y+2z)dxdydz
由对称性，x,y,z积分相同且奇函数在对称区间积分为0
原式=0
