高等数学第十章：重积分 🤯 搞笑版

💡 温馨提示：重积分就像叠罗汉，一层不够就叠两层，两层不够就叠三层！
1. 二重积分的概念和性质 📦

二重积分就是"双重打击"的积分！

定义：

∬Df(x,y)dσ = limλ→0Σf(ξi,ηi)Δσi

几何意义： 曲顶柱体的体积（当f(x,y)≥0时）

物理意义： 平面薄片的质量（f(x,y)是面密度）

性质：

  • 线性性：可以拆开加减
  • 区域可加性：大区域积分=小区域积分和
  • 比较定理：大的函数积分大
  • 中值定理：存在一点使积分=函数值×区域面积

例子： 计算∬D1dσ 就是求区域D的面积！

2. 二重积分的计算方法 🧮

二重积分计算有两大法宝：直角坐标和极坐标！

直角坐标法：

  1. X型区域：先积y后积x

    ∬Df(x,y)dxdy = ∫abdx∫φ₁(x)φ₂(x)f(x,y)dy

  2. Y型区域：先积x后积y

    ∬Df(x,y)dxdy = ∫cddy∫ψ₁(y)ψ₂(y)f(x,y)dx

极坐标法： 当区域是圆或扇形时使用

∬Df(x,y)dxdy = ∬Df(rcosθ,rsinθ)rdrdθ

注意事项：

  • 先画积分区域图！
  • 选择合适的积分次序
  • 极坐标别忘记乘r！
  • 对称性可以简化计算

例子： 计算∬Dxydxdy，D由y=x, y=1, x=0围成

解：∫01dx∫x1xydy = ∫01x(1-x²)/2dx = 1/8

3. 三重积分的定义和性质 📦📦📦

三重积分就是"三倍快乐"的积分！

定义：

∭Ωf(x,y,z)dv = limλ→0Σf(ξi,ηi,ζi)Δvi

性质：

  • 和二重积分类似，有线性性、区域可加性等
  • 物理意义：空间物体的质量（f是体密度）
  • 几何意义：当f≡1时就是区域Ω的体积
🤔 思考：四重积分存在吗？当然存在！只是我们画不出四维空间而已...
4. 三重积分的几何意义和物理意义 🌌

几何意义：

  • 当f(x,y,z)≥0时：四维空间"超体积"（想象不出来很正常）
  • 当f≡1时：空间区域Ω的体积

物理意义：

  • 非均匀物体的质量（f是密度函数）
  • 物体的重心坐标计算
  • 转动惯量计算

例子： 计算半径为R的球体体积：∭Ω1dv = 4/3πR³

5. 三重积分的计算方法🔪✂️📦 🧮🧮

三重积分计算有三种武器：直角坐标、柱坐标和球坐标！

📦直角坐标法：

∭Ωf(x,y,z)dxdydz = ∫abdx∫y₁(x)y₂(x)dy∫z₁(x,y)z₂(x,y)f(x,y,z)dz

1️⃣➡️2️⃣ 先一后二法（投影法）

就像先切蛋糕再分块！先固定x,y，对z积分，再对x,y积分

∭Ωf(x,y,z)dv = ∬Ddxdy∫z₁(x,y)z₂(x,y)f(x,y,z)dz

步骤：

  1. 把三维区域Ω投影到xy平面得D
  2. 确定z的范围：z₁(x,y) ≤ z ≤ z₂(x,y)
  3. 先对z积分，得到关于x,y的函数
  4. 再在D上对x,y积分

例子： 计算∭Ωzdv，Ω由z=xy, y=x, x=1, z=0围成

解：∫01dx∫0xdy∫0xyzdz = 1/48

2️⃣➡️1️⃣ 先二后一法（截面法）

像做CT扫描！先固定z，对x,y积分，再对z积分

∭Ωf(x,y,z)dv = ∫abdz∬D(z)f(x,y,z)dxdy

步骤：

  1. 确定z的范围：a ≤ z ≤ b
  2. 对每个z，求截面区域D(z)
  3. 先在D(z)上对x,y积分
  4. 再对z积分

例子： 计算∭Ω(x²+y²)dv，Ω由z=√(x²+y²)和z=1围成

解：∫01dz∬x²+y²≤z²(x²+y²)dxdy = π/6

🛢️ 柱面坐标法

极坐标的升级版！适合旋转体

x = rcosθ, y = rsinθ, z = z

dv = rdrdθdz

步骤：

  1. 确定r,θ,z的范围
  2. 把被积函数转换为柱坐标形式
  3. 记得乘上r！
  4. 从内到外依次积分

例子： 计算∭Ωzdv，Ω由z=√(x²+y²)和z=1围成

解：∫02πdθ∫01rdr∫r1zdz = π/6

🌐 球面坐标法

地球仪坐标系！适合球体

x = ρsinφcosθ, y = ρsinφsinθ, z = ρcosφ

dv = ρ²sinφdρdφdθ

步骤：

  1. 确定ρ,φ,θ的范围
  2. 把被积函数转换为球坐标形式
  3. 记得乘上ρ²sinφ！
  4. 从内到外依次积分

例子： 计算半径为R的球体体积

解：∫02πdθ∫0πdφ∫0Rρ²sinφdρ = 4/3πR³

6. 重积分的应用 🚀

重积分不只是考试题，还能解决实际问题！

几何应用：

  • 平面区域面积：∬D1dσ
  • 空间区域体积：∭Ω1dv
  • 曲面面积：∬D√(1+zₓ²+z_y²)dxdy

物理应用：

  • 质量：m = ∭Ωρ(x,y,z)dv
  • 重心坐标：x̄ = (∭Ωxρdv)/m
  • 转动惯量：I_z = ∭Ω(x²+y²)ρdv

例子： 求半径为R的均匀半球体的重心

由对称性x̄=ȳ=0，z̄ = (∭zρdv)/V = (3R)/8

第十章模拟试题 📝

1. 判断题 (2分)：二重积分∬Df(x,y)dσ的几何意义是曲顶柱体的体积。

答案：正确（当f(x,y)≥0时）

2. 填空题 (2分)：极坐标下二重积分的面积元素是______。

答案：rdrdθ

3. 选择题 (3分)：计算∬x²+y²≤1√(1-x²-y²)dxdy应使用： A) 直角坐标 B) 极坐标 C) 球坐标 D) 柱坐标

答案：B

4. 计算题 (8分)：计算∬Dxydxdy，D由y=x², y=0, x=1围成。

答案：∫01dx∫0x²xydy = ∫01x⁵/2dx = 1/12

5. 应用题 (10分)：求半径为R的均匀半球体的重心位置。

答案：由对称性x̄=ȳ=0，z̄ = (3R)/8

6. 证明题 (10分)：证明球坐标的体积元素是ρ²sinφdρdφdθ。

答案：计算雅可比行列式|∂(x,y,z)/∂(ρ,φ,θ)| = ρ²sinφ

7. 判断题 (2分)：三重积分∭Ωf(x,y,z)dv=0表示Ω的体积为零。

答案：错误（可能是f的正负部分抵消）

8. 填空题 (2分)：球坐标中，φ的取值范围是______。

答案：[0,π]

9. 选择题 (3分)：计算∭x²+y²+z²≤1zdv应使用： A) 直角坐标 B) 柱坐标 C) 球坐标 D) 任何坐标

答案：C

10. 计算题 (8分)：计算∭Ωzdxdydz，Ω由z=√(x²+y²)和z=1围成。

答案：用柱坐标∫02πdθ∫01rdr∫r1zdz = π/6

11. 计算∬D(x+y)dxdy，其中D由x=0,y=0,x=1,y=1围成

解：∫01dx∫01(x+y)dy = ∫01[xy+y²/2]01dx = ∫01(x+1/2)dx = [x²/2+x/2]01 = 1

12. 计算∬D√(x²+y²)dxdy，其中D为1≤x²+y²≤4

解：极坐标变换，∫02πdθ∫12r·rdr = 2π·[r³/3]12 = 14π/3

13. 计算∭Ωxyzdv，Ω由x=1,y=1,z=1和坐标面围成

解：∫01dx∫01dy∫01xyzdz = ∫01xdx·∫01ydy·∫01zdz = (1/2)·(1/2)·(1/2) = 1/8

14. 计算∭Ωzdv，Ω由z=√(x²+y²)和z=1围成

解：柱坐标，∫02πdθ∫01rdr∫r1zdz = 2π∫01r(1/2-r²/2)dr = π/6

15. 计算半径为R的球体体积

解：球坐标，∫02πdθ∫0πdφ∫0Rρ²sinφdρ = 2π·2·R³/3 = 4πR³/3

16. 计算∫01dy∫y1ex²dx

解：交换积分次序，∫01dx∫0xex²dy = ∫01xex²dx = (e-1)/2

17. 求密度为1的半球体x²+y²+z²≤1, z≥0的质量

解：球坐标，∫02πdθ∫0π/2dφ∫01ρ²sinφdρ = 2π·1·1/3 = 2π/3

18. 计算∬De-(x²+y²)dxdy，D为x²+y²≤a²

解：极坐标，∫02πdθ∫0are-r²dr = 2π·(1-e-a²)/2 = π(1-e-a²)

19. 计算∭Ω(x²+y²)dv，Ω由z=√(x²+y²)和z=1围成

解：柱坐标，∫02πdθ∫01r³dr∫r1dz = 2π·(1/4)·(1/2) = π/4

20. 计算∭Ωz√(x²+y²)dv，Ω由z=0和z=√(1-x²-y²)围成

解：柱坐标，∫02πdθ∫01r²dr∫0√(1-r²)zdz = 2π∫01r²(1-r²)/2dr = π/15
