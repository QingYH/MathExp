1. 向量的数量积 & 向量积 🤼‍♂️

大家好，我是向量小A，这是我的好基友向量小B~

"数量积就像我们的友谊 - 越平行越亲密（值越大），垂直就归零（没共同语言）"

数量积（点积）计算公式：

a·b = |a||b|cosθ = a₁b₁ + a₂b₂ + a₃b₃

记忆技巧： 横乘横 + 竖乘竖 + 深乘深（三维时）

向量积（叉积）计算公式：

a×b = |a||b|sinθ·n = ⎛ i   j   k ⎞ ⎜ a₁ a₂ a₃ ⎟ ⎜ b₁ b₂ b₃ ⎟

记忆技巧： 行列式展开，i(a₂b₃-a₃b₂) - j(a₁b₃-a₃b₁) + k(a₁b₂-a₂b₁)

"叉积就像我们的孩子 - 方向垂直于父母（右手定则），大小看父母夹角（sinθ）"

2. 平面的法向量 & 方程 ✈️

我是平面君，我的法向量n就是我的身份证~

法向量求法：

  1. 已知平面内两个向量 → 叉积得到法向量
  2. 已知三点A,B,C → AB×AC得到法向量

平面方程形式：

点法式：n·(r - r₀) = 0 → A(x-x₀) + B(y-y₀) + C(z-z₀) = 0
一般式：Ax + By + Cz + D = 0
截距式：x/a + y/b + z/c = 1
"我的方程就像我的心情 - 系数A,B,C决定我的倾斜程度，D决定我离原点的距离（脾气大小）"

3. 空间直线的方向向量 & 方程 🚀

我是直线小王子，方向向量s就是我的指南针~

方向向量求法：

  1. 已知两点 → 两点坐标相减
  2. 作为两平面交线 → 两平面法向量的叉积

直线方程形式：

对称式：(x-x₀)/m = (y-y₀)/n = (z-z₀)/p
参数式：x=x₀+mt, y=y₀+nt, z=z₀+pt
一般式：{A₁x+B₁y+C₁z+D₁=0; A₂x+B₂y+C₂z+D₂=0}
"我的参数方程就像我的行程表 - t是我的时间，m,n,p是我的速度分量"

4. 特殊二次曲面 🎪

我们是二次曲面马戏团！来看看我们的绝活~

曲面名称 标准方程 特征
椭球面 x²/a² + y²/b² + z²/c² = 1 三维的椭圆，像个被压扁的球
单叶双曲面 x²/a² + y²/b² - z²/c² = 1 像核电站冷却塔，有"腰身"
双叶双曲面 x²/a² + y²/b² - z²/c² = -1 像两个背对背的喇叭
椭圆抛物面 z = x²/a² + y²/b² 像卫星天线，开口向上
双曲抛物面 z = x²/a² - y²/b² 马鞍形，又被称为"马鞍面"
"我们二次曲面家族聚会时最怕遇到退化情况 - 那会让我们变成平面或者直线，太没面子了！"

5. 空间曲线的方程 🎢

我是空间曲线，比平面曲线多了个z坐标的乐趣~

表示方法：

一般式：{F(x,y,z)=0; G(x,y,z)=0}
参数式：x=x(t), y=y(t), z=z(t)
向量式：r = r(t) = x(t)i + y(t)j + z(t)k

常见空间曲线：

  • 圆柱螺旋线： x=acosθ, y=asinθ, z=bθ
  • 两曲面交线： 如球面与平面的交线是圆
"参数方程是我的DNA，t是我的生命线 - 改变t就能看到我婀娜的身姿变化"

欢乐高数模拟试题 📝

一、判断题（2分×3）

1. 向量a=(1,2,3)和b=(2,4,6)是平行的。

✅ 正确（b=2a）

2. 平面x+y+z=1的法向量是(1,1,0)。

❌ 错误（应该是(1,1,1)）

3. 方程x²+y²=1在空间中表示圆柱面。

✅ 正确（z可以取任意值）

二、填空题（3分×3）

1. 向量a=(1,0,1)和b=(0,1,1)的点积是______。

1×0 + 0×1 + 1×1 = 1

2. 过点(1,2,3)且法向量为(2,-1,1)的平面方程是______。

2(x-1)-(y-2)+(z-3)=0 → 2x-y+z-3=0

3. 直线(x-1)/2=(y+1)/-1=z/3的方向向量是______。

(2,-1,3)

三、选择题（4分×2）

1. 下列哪个是双曲抛物面的方程？
A) x²+y²+z²=1
B) z=x²-y²
C) x²/a²+y²/b²=1
D) x²+y²=z²

B) z=x²-y²（马鞍面）

2. 两平面x+y+z=1和x-y+z=2的交线方向向量是：
A) (1,1,1)
B) (1,-1,1)
C) (2,0,-2)
D) (1,0,1)

C) (2,0,-2)（两法向量(1,1,1)×(1,-1,1)=(2,0,-2)）

四、计算题（10分）

已知三点A(1,0,0), B(0,1,0), C(0,0,1)，求：
(1) 三角形ABC的面积
(2) 过这三点的平面方程

(1) AB=(-1,1,0), AC=(-1,0,1)
AB×AC=(1,1,1) → 面积=½|AB×AC|=½√3
(2) 法向量(1,1,1)，点A代入得：x+y+z=1

五、应用题（12分）

一颗卫星的运行轨迹可以表示为r(t)=(2cost, 2sint, t)，t≥0：
(1) 描述这是什么曲线
(2) 求t=π/2时卫星的位置和速度向量

(1) 圆柱螺旋线（在x²+y²=4的圆柱上螺旋上升）
(2) 位置r(π/2)=(0,2,π/2)
速度v(t)=r'(t)=(-2sint,2cost,1) → v(π/2)=(-2,0,1)
