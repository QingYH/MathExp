1. 平面点集之内点、外点、界点和聚点 📍

大家好，我是平面点集，今天我要介绍我的四个"人格"：

  • 内点：就像宅男，完全不出门，周围全是自己的地盘。数学定义：存在邻域全含于集合内。
  • 外点：社交恐惧症患者，完全不接触集合。数学定义：存在邻域全不含于集合内。
  • 界点：社交达人，一脚门里一脚门外。数学定义：任何邻域都既含集合内点又含外点。
  • 聚点：万人迷，周围永远有人围着。数学定义：任何邻域都含有集合中无限多个点。

例子： 考虑集合A={(x,y)|x²+y²≤1}（单位圆）

(0.5,0.5)是内点，(2,2)是外点，(1,0)是界点，(1,0)也是聚点。

注意：聚点可以属于集合也可以不属于集合！就像明星的粉丝，明星本人可能不在粉丝群里。
2. 多元函数极限的定义和求法 🎯

一元函数极限已经够难了，现在还要多元？别怕，其实原理差不多！

定义： 当点P(x,y)以任何方式趋近于P₀(x₀,y₀)时，f(x,y)都趋近于A。

lim(x,y)→(x₀,y₀) f(x,y) = A

求法技巧：

  1. 先试试直接代入法（如果分母不为零）
  2. 如果直接代入不行，尝试因式分解
  3. 可以尝试极坐标变换：令x=rcosθ, y=rsinθ，然后r→0
  4. 如果不同路径得到不同极限，则极限不存在

例子： 求 lim(x,y)→(0,0) (x²y)/(x⁴+y²)

沿着y=kx路径：极限为0；沿着y=x²路径：极限为1/2 → 极限不存在！

🤔 思考题：为什么多元函数极限比一元难？因为在一元中，点只能从左右两边逼近，而在多元中，点可以从无数条路径逼近！
3. 多元函数连续的定义及判定方法 🌊

连续就是没有"断点"，画图时笔不用抬起来！多元函数连续定义：

lim(x,y)→(x₀,y₀) f(x,y) = f(x₀,y₀)

判定方法：

  • 初等函数在其定义域内都是连续的（就像你每天回家一样自然）
  • 分段函数在分界点处需要单独验证
  • 验证三步走：1) 函数在该点有定义 2) 极限存在 3) 极限等于函数值
注意：多元函数在某点连续，必须要求从所有方向逼近都连续！比一元函数严格多了！

例子： f(x,y) = { (x²y)/(x²+y²) , (x,y)≠(0,0); 0 , (x,y)=(0,0) }

在(0,0)处连续吗？验证极限lim(x,y)→(0,0)(x²y)/(x²+y²)=0=f(0,0)，所以连续。

4. 偏导数的定义和求法 🧩

偏导数就是"偏心"的导数！只对一个变量求导，其他变量当常数。

定义：

∂f/∂x = limΔx→0 [f(x+Δx,y)-f(x,y)]/Δx

∂f/∂y = limΔy→0 [f(x,y+Δy)-f(x,y)]/Δy

求法：

  1. 对x求偏导：把y当常数，然后对x求导
  2. 对y求偏导：把x当常数，然后对y求导
  3. 几何意义：曲面与平行于坐标面的平面的交线的斜率

例子： f(x,y) = x²y + sin(xy)

∂f/∂x = 2xy + ycos(xy)

∂f/∂y = x² + xcos(xy)

😅 常见错误：忘记其他变量是常数！比如∂(xy)/∂x=y，不是y+x(dy/dx)！这里不是隐函数求导！
5. 混合偏导求法 🔄

混合偏导就是先对x求偏导，再对y求偏导，记作fxy或∂²f/∂y∂x

求法：

  1. 先求fx = ∂f/∂x
  2. 再对fx关于y求偏导

重要定理： 如果fxy和fyx都连续，那么它们相等！

例子： f(x,y) = x³y + exy

fx = 3x²y + yexy

fxy = 3x² + exy + xyexy

fy = x³ + xexy

fyx = 3x² + exy + xyexy

确实fxy = fyx！

注意顺序！∂²f/∂x∂y表示先对y求偏导再对x求偏导，与fxy顺序相反！
6. 全微分定义 📈

全微分是多元函数的"最佳线性近似"，就像给曲面贴一个小平面！

定义：

dz = ∂f/∂x dx + ∂f/∂y dy

或者

Δz ≈ fx(x₀,y₀)Δx + fy(x₀,y₀)Δy

几何意义： 用切平面上的增量近似曲面上的增量

例子： f(x,y) = x² + 3xy - y² 在(1,2)处的全微分

fx = 2x + 3y → fx(1,2) = 8

fy = 3x - 2y → fy(1,2) = -1

dz = 8dx - dy

🤓 全微分可以用来估计函数值变化。比如当Δx和Δy很小时，Δz ≈ dz，计算量大大减少！
7. 多元函数可微、连续、偏导存在之间的关系和相应的反例 🔄

这三个概念的关系就像三角恋，复杂得很！

关系图：

可微 ⇒ 连续

可微 ⇒ 偏导存在

但是反过来都不成立！

反例大全：

  • 偏导存在但不连续：f(x,y) = { 0, xy=0; 1, xy≠0 } 在(0,0)处
  • 连续但不可微：f(x,y) = |x| + |y| 在(0,0)处
  • 偏导存在但不可微：f(x,y) = { xy/√(x²+y²), (x,y)≠(0,0); 0, (x,y)=(0,0) }
记住：可微的条件最强！可微要求函数在该点附近可以用一个平面很好地近似，而不仅仅是各个方向有切线。
8. 多元复合函数的求导法则和注意事项 🔗

链式法则来啦！多元复合函数求导就像剥洋葱，一层一层来。

链式法则： 如果z=f(x,y)，而x=x(t)，y=y(t)，那么

dz/dt = ∂f/∂x · dx/dt + ∂f/∂y · dy/dt

更一般的情况：如果z=f(x,y)，而x=x(u,v)，y=y(u,v)，那么

∂z/∂u = ∂f/∂x · ∂x/∂u + ∂f/∂y · ∂y/∂u

∂z/∂v = ∂f/∂x · ∂x/∂v + ∂f/∂y · ∂y/∂v

注意事项：

  1. 画变量关系图，搞清楚谁依赖谁
  2. 不要漏掉任何一条路径
  3. 注意区分∂f/∂x和∂z/∂x（前者是f对第一个变量的偏导，后者是复合后的偏导）

例子： z = e^(x²y), x = uv, y = u+v

∂z/∂u = ∂z/∂x · ∂x/∂u + ∂z/∂y · ∂y/∂u = 2xy e^(x²y) · v + x² e^(x²y) · 1

9. 隐函数求导公式和具体做法 🕵️‍♂️

隐函数就像害羞的函数，不愿意直接表达自己！

单方程情形： F(x,y)=0 确定 y=y(x)

dy/dx = -Fx/Fy

方程组情形： F(x,y,u,v)=0, G(x,y,u,v)=0 确定 u=u(x,y), v=v(x,y)

用雅可比行列式求解：

∂u/∂x = - (∂(F,G)/∂(x,v)) / (∂(F,G)/∂(u,v))

具体步骤：

  1. 确认隐函数存在（Fy≠0）
  2. 两边对x求导（记住y是x的函数）
  3. 解出dy/dx

例子： x² + y² = 1 求 dy/dx

2x + 2y dy/dx = 0 ⇒ dy/dx = -x/y

10. 空间曲线的切线和法平面方程的求法 ✏️

空间曲线就像空中飞舞的彩带，我们来研究它的切线和法平面！

参数方程形式： r(t) = (x(t), y(t), z(t))

切线方向向量：r'(t) = (x'(t), y'(t), z'(t))

切线方程：

(x-x₀)/x'(t₀) = (y-y₀)/y'(t₀) = (z-z₀)/z'(t₀)

法平面方程：（垂直于切线的平面）

x'(t₀)(x-x₀) + y'(t₀)(y-y₀) + z'(t₀)(z-z₀) = 0

例子： 螺旋线 r(t) = (cos t, sin t, t) 在 t=π/4 处

r'(t) = (-sin t, cos t, 1) ⇒ r'(π/4) = (-√2/2, √2/2, 1)

切线方程：(x-√2/2)/(-√2/2) = (y-√2/2)/(√2/2) = (z-π/4)/1

法平面方程：(-√2/2)(x-√2/2) + (√2/2)(y-√2/2) + 1·(z-π/4) = 0

11. 曲面的切平面与法线方程的求法 🧊

曲面就像山坡，切平面就像放在山坡上的平板！

隐式方程 F(x,y,z)=0：

法向量：n = (Fx, Fy, Fz)

切平面方程：

Fx(x₀,y₀,z₀)(x-x₀) + Fy(y-y₀) + Fz(z-z₀) = 0

法线方程：

(x-x₀)/Fx = (y-y₀)/Fy = (z-z₀)/Fz

例子： 球面 x² + y² + z² = 1 在 (1/2, 1/2, √2/2) 处

Fx=2x=1, Fy=2y=1, Fz=2z=√2

切平面：1·(x-1/2) + 1·(y-1/2) + √2(z-√2/2) = 0

法线：(x-1/2)/1 = (y-1/2)/1 = (z-√2/2)/√2

12. 方向导数定义及求法 🧭

方向导数告诉你函数在某个方向上的变化率！

定义：

Duf(x₀,y₀) = limh→0 [f(x₀+hu₁, y₀+hu₂) - f(x₀,y₀)] / h

其中 u = (u₁, u₂) 是单位方向向量

计算公式：

Duf = fxu₁ + fyu₂ = ∇f · u

例子： f(x,y) = x² + y² 在 (1,1) 沿 (1,1) 方向的方向导数

∇f = (2x, 2y) = (2,2) 在 (1,1)

单位化 u = (1/√2, 1/√2)

Duf = 2·(1/√2) + 2·(1/√2) = 4/√2 = 2√2

13. 函数梯度的定义、求法和意义 📶

梯度就像指南针，指向函数增长最快的方向！

定义：

∇f = (fx, fy, fz)

性质：

  • 方向导数最大值 = ||∇f||，方向为梯度方向
  • 梯度垂直于等高线
  • 梯度指向函数值增长最快的方向

例子： f(x,y) = x²y

∇f = (2xy, x²)

在(1,2)处：∇f = (4,1)，方向导数最大值为√(16+1)=√17

14. 多元函数极值的求法，拉格朗日乘数法求条件极值 🏔️

找极值就像在山里找最高点和最低点，只不过现在是多维的！

无条件极值（自由极值）：

  1. 先求偏导，找驻点：

    fx = 0, fy = 0

  2. 用二阶导数判别法：

    计算判别式：

    D = fxxfyy - (fxy)²

    • D > 0 且 fxx > 0 → 极小值（像碗底）
    • D > 0 且 fxx < 0 → 极大值（像山顶）
    • D < 0 → 鞍点（像马鞍，既不是最高也不是最低）
    • D = 0 → 无法判断，需要其他方法

例子： 求 f(x,y) = x³ + y³ - 3xy 的极值

fx = 3x² - 3y = 0

fy = 3y² - 3x = 0

解得驻点：(0,0) 和 (1,1)

在(0,0): D = (-3)(-3) - (-3)² = 0 → 无法判断

在(1,1): D = (6)(6) - (-3)² = 27 > 0 且 fxx=6>0 → 极小值

拉格朗日乘数法（条件极值）：

当有约束条件 g(x,y)=0 时，就像戴着镣铐跳舞！

  1. 构造拉格朗日函数：

    L(x,y,λ) = f(x,y) - λg(x,y)

  2. 求偏导并解方程组：

    Lx = 0, Ly = 0, Lλ = 0

  3. 判断极值性质（通常根据实际问题判断）

例子： 在 x² + y² = 1 条件下，求 f(x,y) = xy 的最大值

构造 L = xy - λ(x² + y² - 1)

Lx = y - 2λx = 0

Ly = x - 2λy = 0

Lλ = -(x² + y² - 1) = 0

解得：x = y = ±√2/2，最大值为 f(√2/2, √2/2) = 1/2

🤓 拉格朗日乘数法中的λ可以理解为"约束条件的价格" - 它告诉你如果放松一点约束，目标函数能改善多少！

实际应用：

  • 经济学：预算约束下的效用最大化
  • 工程学：材料约束下的最优设计
  • 物理学：约束系统的平衡位置

第九章模拟试题 📝

1. 判断题 (2分)：若函数在某点可微，则在该点一定连续且偏导数存在。

答案：正确

2. 填空题 (2分)：函数z = ln(x² + y²)在点(1,1)处的全微分dz = ______。

答案：dx + dy

3. 选择题 (3分)：下列哪个函数在(0,0)点连续但不可微？ A) x² + y² B) |x| + |y| C) xy/(x²+y²) D) x³y

答案：B

4. 计算题 (8分)：求函数f(x,y) = x³ + y³ - 3xy的极值。

答案：驻点(0,0)和(1,1)；(0,0)是鞍点，(1,1)是极小值点

5. 应用题 (10分)：用拉格朗日乘数法求表面积为6的长方体的最大体积。

答案：设长宽高为x,y,z，约束2(xy+yz+zx)=6，构造L=xyz-λ(2xy+2yz+2zx-6)，解得x=y=z=1，最大体积为1

6. 证明题 (10分)：证明若f(x,y)在点P₀可微，则f在P₀沿任何方向的方向导数都存在。

答案：由可微定义，Δf = fxΔx + fyΔy + o(ρ)，方向导数Duf = lim(f(P₀+tu)-f(P₀))/t = fxu₁ + fyu₂

7. 判断题 (2分)：混合偏导数fxy和fyx在任何情况下都相等。

答案：错误（需要连续条件）

8. 填空题 (2分)：曲面z = x² + y²在点(1,1,2)处的切平面方程是______。

答案：2(x-1) + 2(y-1) - (z-2) = 0

9. 选择题 (3分)：函数f(x,y)在点P₀沿哪个方向的方向导数最大？ A) (1,0) B) (0,1) C) 梯度方向 D) 等高线切线方向

答案：C

10. 计算题 (8分)：设z = f(xy, x²+y²)，其中f具有二阶连续偏导数，求∂²z/∂x∂y。

答案：设u=xy, v=x²+y²，则∂z/∂x = f₁·y + f₂·2x，再对y求导得∂²z/∂x∂y = f₁ + y(f₁₁·x + f₁₂·2y) + 2x(f₂₁·x + f₂₂·2y)

11. 应用题 (10分)：求点(1,2,3)到平面x + y + z = 1的最短距离。

答案：构造距离平方函数D²=(x-1)²+(y-2)²+(z-3)²，约束x+y+z=1，用拉格朗日乘数法解得最短距离为√14

12. 证明题 (10分)：设u = f(x,y)可微，且在极坐标x=rcosθ, y=rsinθ下与θ无关，证明y∂u/∂x - x∂u/∂y = 0。

答案：由题意∂u/∂θ=0，而∂u/∂θ=(∂u/∂x)(-rsinθ)+(∂u/∂y)(rcosθ)=-y∂u/∂x+x∂u/∂y=0

13. 判断题 (2分)：若函数在某点的所有方向导数都存在，则函数在该点可微。

答案：错误

14. 填空题 (2分)：曲线x=t, y=t², z=t³在t=1处的切线方程是______。

答案：(x-1)/1 = (y-1)/2 = (z-1)/3

15. 选择题 (3分)：用拉格朗日乘数法求极值时，乘数λ的几何意义是： A) 目标函数的增长率 B) 约束条件的"价格" C) 极值点的高度 D) 切平面的斜率

答案：B
